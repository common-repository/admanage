在网站底部，页面头部和页面底部添加信息

== Description ==
这个插件能够让用户不用修改主题页面，就能添加信息到网站底部，页面底部和页面头部，可以把广告添加到页面头部,页面左上交，页面右上角和页面底部，也可以把版权声明添加到页面底部。可以把备案，统计代码和各种声明添加到网站底部
